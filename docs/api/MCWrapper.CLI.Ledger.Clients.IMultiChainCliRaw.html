SignRawTransactionAsync(String, Object[], Object[], String)

Sign inputs for raw transaction (serialized, hex-encoded).

The second optional argument (may be null) is an array of previous transaction outputs that this transaction depends on but may not yet be in the block chain.

The third optional argument (may be null) is an array of base58-encoded private keys that, if given, will be the only keys used to sign the transaction.

Requires wallet passphrase to be set with walletpassphrase call.

Blockchain name is inferred from CliOptions properties.

Declaration
Task<CliResponse<SignRawTransactionResult>> SignRawTransactionAsync(string tx_hex, object[] prevtxs, object[] privatekeys, string sighashtype)
Parameters
Type Name Description
System.String tx_hex

The transaction hex string
System.Object[] prevtxs

An json array of previous dependent transaction outputs
System.Object[] privatekeys

A json array of base58-encoded private keys for signing
System.String sighashtype

The signature hash type. Must be one of: "All", "NONE", "SINGLE", "ALL|ANYONECANPAY", "NONE|ANYONECANPAY", "SINGLE|ANYONECANPAY"
Returns
